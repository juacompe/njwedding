เว็บไซต์นี้เป็นเว็บที่จะคอย feed ทวีตที่ถูก tag ด้วย #thaiflood มาแสดงโดยจะกรองทวีตที่ขึ้นต้นด้วย RT ออกไป (ไม่งั้นอ่านไม่ทัน -___-")

ข้อจำกัด

หน้าเว็บจะ refresh เองโดยอัตโนมัติทุกๆ 5 นาทีเพื่อดึงรูปใหม่ๆมา สามารถกด refresh เองก็ได้ (แต่ไม่แนะนำนะครับ เพราะ server มันตัวเล็กๆ ที่เราเช่นกันเอง ถ้าทุกคนรุม refresh มันอาจจะ down ได้ -/\-)

server จะดึงทวีตใหม่มาทีละ 10 ทวีตทุกๆ 1 นาที ถ้านาทีไหนมีคนทวีตกันเยอะๆ บางรูปอาจจะไม่ถูกดึงมานะครับ (ไม่กล้าดึงเยอะกว่านี้ กลัว server process ไม่ไหว)

pic.twitter.com (เพราะมันงก ไม่ยอมเปิด API -*-)

วิธีทำให้รูปที่ทวีตมี link ไปยัง Google Map

รูปที่ถูกทวีต และมี geo location แนบมา (ต้อง enable location service ใน twitter client ที่ใช้ในมือถือก่อน) จะมี link ไปยัง Google map ให้ ทำให้สามารถกดดูได้ว่ารูปที่ถูกถ่าย ถูกถ่ายจากตำแหน่งใด

โปรแกรม Thaiflood

ถ้าใช้โปรแกรม Thaiflood ในมือถือ แล้วเลือก "ถ่ายภาพและส่งไปยัง Twitter" จะมาทั้งรูปและ location โดยอัตโนมัตินะครับ
